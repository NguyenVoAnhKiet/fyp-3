# docs/project-report/

## Responsibility

Contains the drafts, outlines, and formatting rules for the final project report. The markdown files here are designed to be easily copy-pasted into Microsoft Word with minimal formatting loss.

## Files

- `format-rules.md` — Formatting constraints (headers, lists, bold/italic) to ensure Word compatibility. No HTML/CSS allowed.
- `outline.md` — High-level outline for all chapters.
- `chuong-1.md` — Detailed drafted content for Chapter 1.
- `chuong-2.md` — Detailed drafted content for Chapter 2 (Cơ sở lý thuyết).
- `chuong-3.md` — Detailed drafted content for Chapter 3 (Phân tích và thiết kế hệ thống).
- `chuong-4.md` — Detailed drafted content for Chapter 4 (Triển khai và đánh giá).
- `chuong-5.md` — Detailed drafted content for Chapter 5 (Kết luận và hướng phát triển).
- `presentation-format.md` — University thesis/dissertation formatting guide (layout structure, font/spacing specs, cover pages, heading numbering, figure/table/citation rules).
