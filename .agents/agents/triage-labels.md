# Triage Labels

The following labels are used in the frontmatter of issue files to track state.

| Role | Label |
|------|-------|
| needs-triage | `needs-triage` |
| needs-info | `needs-info` |
| ready-for-agent | `ready-for-agent` |
| ready-for-human | `ready-for-human` |
| wontfix | `wontfix` |
